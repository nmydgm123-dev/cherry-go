# 🚀 CherryGo 超级快速上手指南

## 三步运行起来！

---

### 第一步：安装微信开发者工具

1. 下载并安装：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
2. 安装后用微信扫码登录
3. 选择"小程序"项目类型

---

### 第二步：导入项目

在微信开发者工具中：

1. 点击「+」或者「导入项目」
2. **项目目录**：选择 `C:\Users\19157\Desktop\CherryGo`
3. **AppID**：选择您的小程序ID（wxa57dbca9d15c9e2b）
4. **项目名称**：填写「CherryGo」
5. 点击「新建」或「导入」

---

### 第三步：开通云开发（免费）

这是**最关键的一步**：

#### 1. 打开云开发控制台
- 在微信开发者工具左上角，点击「云开发」按钮
- 如果提示未开通，点击「开通」
- 选择「免费版」就可以了！

#### 2. 创建数据库集合
打开云开发控制台 -> 数据库 -> 点击「+」创建以下集合：

```
users
products
orders
shipping_rules
coupons
invites
farms
```

每个集合名称都要**完全一致**！

---

### 第四步：部署云函数

在微信开发者工具中：

1. 在左侧文件树，找到 `src/cloudfunctions/` 文件夹
2. 右键点击每个子文件夹（如 login、createOrder 等）
3. 选择「上传并部署：云端安装依赖」
4. 等待进度条完成（每个云函数都要上传）

**要上传的云函数：**
- login
- getProducts
- createOrder
- getOrders
- getOrder
- shipOrder
- queryLogistics
- getStatistics
- saveProduct
- searchOrder

---

### 第六步：设置自己为果园所有者

1. 先在小程序里运行一次（随便点一下）
2. 打开云开发控制台 -> 数据库 -> `users` 集合
3. 找到刚才自动创建的您的记录
4. 点击编辑，把 `role` 字段改为 `owner`
5. 保存后重新打开小程序

## ❓ 常见问题

### Q: 云函数上传失败
A: 检查网络，重试一次，或者分开一个个上传

### Q: 数据库权限问题
A: 云开发控制台 -> 数据库 -> 每个集合 -> 权限设置 -> 选择「所有用户可读，仅创建者可写」

### Q: 图片上传失败
A: 云开发控制台 -> 云存储 -> 权限设置 -> 改为「所有用户可读写」
