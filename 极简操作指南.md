4. **AppID选择**：选择您的小程序（wxa57dbca9d15c9e2b）
5. **项目名称**：填「CherryGo」

## ☁️ 第二步：开通云开发（30秒）

1. **在微信开发者工具左上角，点击「云开发」按钮**
   - 如果第一次点，会提示开通
   - 选择「免费版」
   - 随便填个环境名称（如「CherryGo开发」）
   - 点「开通」
   - 等待10秒钟...

✅ 看到云开发控制台就成功了！

---

## 📊 第三步：一键初始化数据库

### 方法A：用云函数一键初始化（推荐）

1. **在左侧文件树中找到**：`src/cloudfunctions/initDB`
2. **右键点击** `initDB` 文件夹
3. **选择「上传并部署：云端安装依赖」**
4. **等待上传完成**
5. **打开云开发控制台 -> 云函数**
6. **找到 initDB，点击「测试」**
7. **点击「调用测试」按钮**
8. **看到返回成功就完成了！**

### 方法B：手动添加（如果A不行）

打开云开发控制台 -> 数据库 -> 点击「+」创建集合：

```
✅ users
✅ products
✅ orders
✅ shipping_rules
✅ coupons
✅ invites
✅ farms
```

然后参考 `快速上手.md` 添加测试数据。

## 🔐 第五步：设置数据库权限

**非常重要！必须设置！**

在云开发控制台 -> 数据库：

1. 点击每个集合名称（如 `users`）
2. 点击「权限设置」
3. 选择「**所有用户可读，仅创建者可写**」
4. 点击「确定」
5. **对所有7个集合都做同样的设置！**

---

## 🛍️ 第六步：设置自己为果园所有者

1. **先在小程序预览里随便点一下**
2. **打开云开发控制台 -> 数据库 -> users 集合**
3. **找到您的记录**
4. **点击「编辑」**
5. **找到 `role` 字段，把值改成 `"owner"`**
6. **保存**
7. **重新打开小程序**

✅ 现在您就是果园主了！可以看到所有管理功能！

## ❓ 遇到问题？

1. **云函数上传失败**：网络问题，重试一下
2. **数据库权限错误**：回去检查第五步
3. **看不到管理功能**：检查第六步 `role` 是否为 `owner`
