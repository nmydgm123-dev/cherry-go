## 📱 第一步：导入项目（在微信开发者工具中）

- 打开您已经安装的 **微信开发者工具**

### 2️⃣ 导入项目
1. 点击 **「导入项目」** 或 **「+」** 按钮
2. **项目目录** 选择：`C:\Users\19157\Desktop\CherryGo\miniprogram`
3. **AppID** 选择：`wxa57dbca9d15c9e2b`（您的小程序AppID）
4. **项目名称** 填：`CherryGo`
5. **开发模式** 选择：`小程序`
6. **后端服务** 选择：`微信云开发`
7. 勾选 **「不使用云服务」**（先勾选，后面我们再开启云开发）
8. 点击 **「新建」** 或 **「导入」**

## ☁️ 第二步：开通云开发（非常重要！）

### 1️⃣ 开启云开发
1. 在微信开发者工具顶部，点击 **「云开发」** 按钮
2. 如果提示未开通，点击 **「开通」**
3. 选择 **「免费版」** 或 **「按量付费版」**（推荐免费版）
4. 填写环境名称（如：`cherrygo-dev`）
5. 点击 **「确认开通」**
6. 等待 1-2 分钟...

### 2️⃣ 创建数据库集合
在云开发控制台（点击顶部「云开发」进入）中：

1. 点击左侧菜单 **「数据库」**
2. 点击 **「+」** 添加以下集合（注意名称要完全一致！）：

```
✅ users
✅ products
✅ orders
✅ shipping_rules
✅ coupons
✅ invites
✅ farms
```

点击每个集合名称，再点击 **「权限设置」**，选择 **「所有用户可读，仅创建者可写」**

### 3️⃣ 一键初始化数据（最简单！）
1. 在微信开发者工具左侧文件树中，找到 `cloudfunctions` 文件夹
2. 右键点击 `initDB` 文件夹
3. 选择 **「上传并部署：云端安装依赖」**
4. 等待上传完成（看到控制台提示「上传成功」）
5. 在云开发控制台点击 **「云函数」**
6. 找到 `initDB` 云函数
7. 点击 **「测试」** 按钮
8. 点击 **「调用测试」** 按钮
9. 看到返回结果为 `"code": 0` 说明成功！

现在您已经有测试数据了！

## 🚀 第三步：上传所有云函数

在微信开发者工具左侧文件树中：

1. 依次右键点击以下云函数文件夹：
   - `login`
   - `getProducts`
   - `getOrders`
   - `getOrder`
   - `createOrder`
   - `shipOrder`
   - `getStatistics`

2. 对 **每个** 云函数执行：
   - 右键点击文件夹
   - 选择 **「上传并部署：云端安装依赖」**
   - 等待进度条完成

---

## 👑 第四步：设置您为果园所有者（获得所有权限）

### 1️⃣ 先在小程序里点击一下
1. 在微信开发者工具右侧的模拟器中，点击 **「我的」** 页面
2. 点击 **「点击登录」**
3. 允许授权（需要扫码）

### 2️⃣ 然后修改数据库
1. 回到云开发控制台 → 数据库 → `users` 集合
2. 找到您刚才登录创建的用户记录
3. 点击 **「编辑」** 按钮
4. 找到 `role` 字段，把值从 `"browser"` 改为 `"owner"`
5. 点击 **「更新」** 或 **「保存」**

### 3️⃣ 重新打开小程序
在微信开发者工具中，点击顶部 **「编译」** 按钮刷新

现在您是果园所有者了！可以看到所有管理功能！

## ❓ 常见问题

### Q: 云函数上传失败怎么办？
A: 检查网络连接，或者分开单个单个上传

### Q: 看不到管理功能？
A: 确认您在数据库中的 `role` 是 `"owner"`，然后重新编译小程序

### Q: 找不到云开发按钮？
A: 确保在微信开发者工具中，您导入的是正确的项目（miniprogram 目录）
